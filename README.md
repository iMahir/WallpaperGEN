### Installation

1. Clone the repository

2. Install the required dependencies from `requirements.txt`:

   ```bash
   pip install -r requirements.txt
   ```
   
3. Change the color and dimensions in `main.py`

4. Run main.py:
   ```bash
   python3 main.py
   ```
